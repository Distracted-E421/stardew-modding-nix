# Custom Stardew Valley Modpack Plan

**Target Version**: Stardew Valley 1.6.15
**SMAPI Version**: Latest compatible with 1.6.15
**Harmony Version**: 2.x+
**Profile**: Full Mega Expanded (500+ mods)

## Design Goals

1. **Stability First**: Prioritize proven compatibility over bleeding-edge features
2. **Performance**: Keep load times under 90 seconds
3. **Cohesion**: Mods should feel like a unified experience
4. **Maintainability**: Document everything for future updates
5. **Shareability**: Easy distribution via Phoenix web app

## Testing Checkpoints

### After Phase 1 (Frameworks)
- [ ] Game launches without errors
- [ ] SMAPI log shows all frameworks loaded
- [ ] GMCM menu accessible

### After Phase 2 (Expansions)
- [ ] All new areas accessible
- [ ] New NPCs have schedules
- [ ] No map errors in log

### After Phase 3 (UI)
- [ ] UI elements not overlapping
- [ ] All keybinds work
- [ ] Map shows all NPCs

### After Phase 4 (Mechanics)
- [ ] Crafting works correctly
- [ ] Farming mechanics intact
- [ ] Combat feels responsive

### After Phase 5-6 (Content)
- [ ] NPCs have dialogue
- [ ] Events trigger correctly
- [ ] New items craftable

### Final Testing
- [ ] Complete one in-game year
- [ ] Visit all new areas
- [ ] Befriend at least one new NPC
- [ ] No crashes or freezes
- [ ] Load time acceptable (<90s)
